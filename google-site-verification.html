# Google Search Console 소유권 확인 파일 # 이 파일은 Google에서 사이트 소유권을 확인하기 위해 사용됩니다. # # 사용 방법: # 1. Google Search Console (https://search.google.com/search-console)에 접속 # 2. 속성 추가 → URL 접두어 선택 # 3. https://helperseries.github.io/knowit/ 입력 # 4. 소유권 확인 방법 중 "HTML 파일 업로드" 선택 # 5. Google이 제공하는 파일명을 이 파일명으로 변경하여 업로드 # # 예시: google1234567890abcdef.html # # 참고: 이 파일은 실제 Google에서 제공하는 파일로 교체해야 합니다.
